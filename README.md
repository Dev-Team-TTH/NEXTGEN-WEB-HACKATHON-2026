# NEXTGEN-WEB-HACKATHON-2026

Nhánh main: Chỉ chứa code đã hoàn thiện, chạy ổn định (để demo cho giám khảo).

Nhánh dev (hoặc develop): Tạo nhánh này từ main. Đây là nơi tập kết code của cả nhóm.

Nhánh tính năng (Feature): Khi một thành viên làm chức năng mới (ví dụ: login), họ sẽ:

Tạo nhánh mới từ dev: git checkout -b feature/login

Code và commit trên nhánh này.

Khi xong, đẩy lên GitHub: git push origin feature/login
